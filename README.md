## About

This android app get data from <a href="https://www.edamam.com/">edamam</a> API

## Features

Get random or custom food for every week day.<br>
You can choose different food for every day.<br>
A recipe can be found for each dish.

- Mon: Chicken
- Tue: Carrot
- Wen: Chili
- Thu: Tomato
- Fri: Avocado
